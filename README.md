# mxalias community

The public home for **[mxalias](https://mxalias.com)** — a privacy-first email alias & forwarding service.

## 💡 Feature requests

Have an idea? **[Browse and vote on feature requests →](https://github.com/mxalias/community/discussions/categories/feature-requests)**

- **Search first** — if your idea already exists, give it an **upvote** instead of opening a duplicate.
- Sort by **Top** to see what the community most wants.
- Status is tracked with labels: `planned` · `in-progress` · `shipped`.
- See the pinned **“How this works + Roadmap”** post in [Announcements](https://github.com/mxalias/community/discussions/categories/announcements).

You can reach the same board from inside the app — the **Feature requests** link in the dashboard sidebar and the 💡 button in the browser-extension popup.

## 🧩 Browser extension

This is also the public home for the mxalias browser extension — releases and changelog are published here.
